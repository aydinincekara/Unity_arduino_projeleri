Eğitim Modülü 01

Temel Bağlantı ve Mimari Altyapı

Unity ve Arduino donanımlarını birbiriyle konuşturmanın en standart ve kararlı yolu Seri Port (Serial Communication) üzerinden haberleşme sağlamaktır. Bu sistem mimarisinde Arduino sensör verilerini okuyup paketler halinde bilgisayara gönderirken; Unity bu portu sürekli dinleyerek gelen verileri eş zamanlı olarak çözümler.

1. Yapılandırma Adımları

  1. Unity API Uyumluluk Ayarı: Unity varsayılan olarak optimize edilmiş hafif bir .NET profili kullanır. Seri haberleşme kütüphanesini (System.IO.Ports) aktif etmek için Unity editöründe üst menüden sırasıyla Edit -> Project Settings -> Player adımlarını takip edin. Açılan panelde Other Settings altında bulunan Api Compatibility Level ayarını .NET Framework (veya yeni sürümlerde .NET Standard 2.1) olarak değiştirin.
  2. Arduino Tarafında Veri Paketleme: Birden fazla sensör girdisi olduğunda verilerin birbirine karışmaması için değerlerin arasına ayırt edici bir işaret (örneğin virgül) eklenmeli ve her paket satır sonu karakteri (Serial.println) ile bitirilmelidir.

2. Örnek Donanım (C++) Kodu

Arduino'ya bağlanmış bir potansiyometre ve buton verisini paketleyip seri porta yazdıran temel taslak:

void setup() {
  Serial.begin(9600); // Unity ile baud hızı eşleşmelidir.
  pinMode(2, INPUT_PULLUP); // 2 numaralı pinde buton
}

void loop() {
  int sensorDeger = analogRead(A0);
  int butonDurum = digitalRead(2);

  // Verileri virgülle ayırarak tek satırda gönderiyoruz
  Serial.print(sensorDeger);
  Serial.print(",");
  Serial.println(butonDurum); // \n satır sonu tetiklenir

  delay(20); // ~50Hz veri yenileme frekansı
}

3. Örnek Unity Altyapı (C#) Kodu

Aşağıdaki C# betiği belirtilen portu arka planda veya ana döngüde dinleyerek verileri ayıklar:

using System;
using System.IO.Ports;
using UnityEngine;

public class ArduinoHaberlesme : MonoBehaviour {
    private SerialPort stream;
    public string portAdi = "COM3"; // Donanımınızın bağlı olduğu port

    void Start() {
        stream = new SerialPort(portAdi, 9600);
        stream.ReadTimeout = 50;
        try { stream.Open(); }
        catch (Exception e) { Debug.LogError("Bağlantı hatası: " + e.Message); }
    }

    void Update() {
        if (stream != null && stream.IsOpen) {
            try {
                string gelenVeri = stream.ReadLine();
                if (!string.IsNullOrEmpty(gelenVeri)) {
                    string[] parcalar = gelenVeri.Split(',');
                    if (parcalar.Length >= 2) {
                        int potDeğeri = int.Parse(parcalar[0]);
                        int butonDurumu = int.Parse(parcalar[1]);
                        // Verileri projenizde özgürce kullanın
                    }
                }
            }
            catch (TimeoutException) { /* Veri akışı bekleniyor */ }
        }
    }

    void OnApplicationQuit() {
        if (stream != null && stream.IsOpen) stream.Close();
    }
}
