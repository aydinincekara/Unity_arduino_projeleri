Proje 6: Fiziksel Renk, Dijital Yansıma

TCS3200 Renk Sensörü ile Unity Sahne Renklendirme

🎯 Proje Hakkında

Bu projede, fiziksel bir objenin rengini bir "göz" gibi algılayan TCS3200 renk sensörünü kullanacağız. Sensörün önüne hangi renkte bir nesne tutarsanız, Arduino bu nesnenin RGB (Red, Green, Blue) bileşenlerini analiz edip sayısal değerlere dönüştürecek ve Unity'deki sanal nesneye iletecek.

💡 Hap Bilgi: TCS3200 sensörü ışığın yoğunluğunu frekansa dönüştürür. Sensör üzerindeki S2 ve S3 pinlerini kullanarak Kırmızı, Yeşil veya Mavi filtrelerini seçeriz ve o rengin yoğunluğunu Arduino ile okuruz.

🛠️ Gerekli Donanımlar ve Yazılımlar

  • 1x Arduino UNO / Nano
  • 1x TCS3200 Renk Sensörü
  • Jumper Kablolar (Dişi-Erkek)

Adım 1: Arduino Kodu

#define S0 2
#define S1 3
#define S2 4
#define S3 5
#define sensorOut 6

void setup() {
  pinMode(S0, OUTPUT); pinMode(S1, OUTPUT);
  pinMode(S2, OUTPUT); pinMode(S3, OUTPUT);
  pinMode(sensorOut, INPUT);
  
  // Frekans ölçekleme (20%)
  digitalWrite(S0, HIGH); digitalWrite(S1, LOW);
  Serial.begin(9600);
}

void loop() {
  // Sadece Kırmızı örneği (S2 ve S3 ile filtreleme)
  digitalWrite(S2, LOW); digitalWrite(S3, LOW);
  int red = pulseIn(sensorOut, LOW);
  
  // (Yeşil ve Mavi için filtre ayarlarını değiştirip seri porta gönder)
  Serial.println(red); // Basitleştirilmiş örnek
  delay(100);
}
