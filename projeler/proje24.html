Proje 24: Sınırları Aşmak (ESP-NOW)

İnternetsiz ve Modemsiz P2P Haberleşme ile Kablosuz Araç Kontrolü

🎯 Proje Hakkında

Wi-Fi ağlarının zayıf çektiği açık alanlarda, sergilerde veya fuarlarda cihazlarınızı nasıl yöneteceğinizi hiç düşündünüz mü? Espressif tarafından geliştirilen ESP-NOW protokolü sayesinde iki ESP32 kartı, tıpkı kablosuz mouse ve klavyeler gibi birbiriyle doğrudan (Peer-to-Peer) eşleşir. Sistemin çalışması için ne bir internete, ne de bir modeme ihtiyaç vardır.


💡 Hap Bilgi (MAC Adresi): ESP-NOW ile veri gönderebilmek için "Alıcı" ESP32'nin MAC Adresini (Tıpkı bir telefon numarası gibi cihazın eşsiz kimliği) "Verici" ESP32'nin koduna yazmamız gerekir.

🛠️ Gerekli Donanımlar ve Yazılımlar

Donanım Listesi

  • 2x ESP32 Geliştirme Kartı (Verici ve Alıcı)
  • 1x SG90 Servo Motor (Direksiyon Simülasyonu için)
  • Jumper Kablolar

Yazılım ve Kütüphaneler

  • Arduino IDE: esp_now.h ve WiFi.h (ESP32 kart paketiyle gömülü gelir).
  • Unity: Standart System.IO.Ports kütüphanesi (Sadece PC'ye bağlı verici ESP32 ile haberleşmek için).

⚙️ Nasıl Yapılır? (Adım Adım)

⚠️ ÖNEMLİ: Kodları yüklemeden önce "Alıcı" ESP32'nin MAC adresini bulmalısınız. Alıcı karta basit bir "WiFi.macAddress()" okuma kodu yükleyerek Serial Monitörden adresi not alın (Örn: 24:6F:28:AE:5C:20).

Adım 1: VERİCİ (Master) ESP32 Kodu (Bilgisayara Takılacak Olan)

Bu kart USB ile bilgisayara bağlıdır. Unity'den "A" veya "D" komutlarını okur ve saliseler içinde havadaki Alıcı ESP32'ye gönderir.

#include <esp_now.h>
#include <WiFi.h>

// ALICI ESP32'NİN MAC ADRESİNİ BURAYA YAZIN
uint8_t aliciAdresi[] = {0x24, 0x6F, 0x28, 0xAE, 0x5C, 0x20};

// Gönderilecek Veri Paketi (Yapı - Struct)
typedef struct struct_mesaj {
  char komut;
} struct_mesaj;

struct_mesaj gidenVeri;
esp_now_peer_info_t peerInfo;

void setup() {
  Serial.begin(115200);
  
  // Wi-Fi'ı Station moduna al ama ağa bağlanma
  WiFi.mode(WIFI_STA);

  // ESP-NOW Başlat
  if (esp_now_init() != ESP_OK) {
    Serial.println("ESP-NOW Baslatilamadi!");
    return;
  }

  // Alıcıyı kaydet
  memcpy(peerInfo.peer_addr, aliciAdresi, 6);
  peerInfo.channel = 0;  
  peerInfo.encrypt = false;
  esp_now_add_peer(&peerInfo);
}

void loop() {
  // Unity'den veri gelirse oku
  if (Serial.available() > 0) {
    gidenVeri.komut = Serial.read();
    
    // Alıcı karta ESP-NOW üzerinden yolla
    esp_err_t sonuc = esp_now_send(aliciAdresi, (uint8_t *) &gidenVeri, sizeof(gidenVeri));
  }
}

Adım 2: ALICI (Slave) ESP32 Kodu (Araca/Robota Takılacak Olan)

Bu kart tamamen bağımsızdır, gücünü pilden alır. Havadan gelen komutu okuyup aracı (servoyu) sağa sola yönlendirir.

#include <esp_now.h>
#include <WiFi.h>
#include <ESP32Servo.h>

Servo direksiyon;
const int servoPin = 26;

// Gelecek Veri Paketi (Vericideki ile birebir aynı olmalı)
typedef struct struct_mesaj {
  char komut;
} struct_mesaj;

struct_mesaj gelenVeri;

// Veri alındığında otomatik çalışacak fonksiyon
void veriGeldi(const uint8_t * mac, const uint8_t *gelenData, int uzunluk) {
  memcpy(&gelenVeri, gelenData, sizeof(gelenVeri));
  
  // 'A' gelirse sola dön (45 derece)
  if(gelenVeri.komut == 'A') direksiyon.write(45);
  // 'D' gelirse sağa dön (135 derece)
  else if(gelenVeri.komut == 'D') direksiyon.write(135);
  // Diğer durumlarda düz git (90 derece)
  else direksiyon.write(90);
}

void setup() {
  Serial.begin(115200);
  direksiyon.attach(servoPin);
  direksiyon.write(90);

  WiFi.mode(WIFI_STA);
  if (esp_now_init() != ESP_OK) {
    Serial.println("ESP-NOW Hatasi");
    return;
  }

  // Gelen verileri dinlemeye başla
  esp_now_register_recv_cb(veriGeldi);
}

void loop() {
  // ESP-NOW arka planda çalıştığı için loop() boş kalabilir
}

Adım 3: Unity Tarafında İletişim

Unity sadece bilgisayara takılı olan "Verici" karta normal seri port (SerialPort) komutları yollayacaktır. ESP-NOW kısmı tamamen donanımsaldır, oyun motorunu yormaz.

using UnityEngine;
using System.IO.Ports;

public class EspNowKontrol : MonoBehaviour {
    SerialPort port = new SerialPort("COM3", 115200); // ESP32 hızına (115200) dikkat!

    void Start() {
        if(!port.IsOpen) port.Open();
    }

    void Update() {
        if (port.IsOpen) {
            // Klavyeden A'ya basılırsa
            if (Input.GetKey(KeyCode.A)) {
                port.Write("A");
            }
            // Klavyeden D'ye basılırsa
            else if (Input.GetKey(KeyCode.D)) {
                port.Write("D");
            }
            // Hiçbirine basılmazsa düz (Sıfırla) komutu
            else {
                port.Write("S"); 
            }
        }
    }

    void OnApplicationQuit() {
        if(port.IsOpen) {
            port.Write("S");
            port.Close();
        }
    }
}

🚀 Geleceğe Yatırım ve Optimizasyon Önerileri

✨ Mimari Uyumluluk (Motor Değişimi): Oluşturduğunuz bu kablosuz ağ yapısı, oyun motorundan tamamen bağımsızdır. Yakın gelecekte öğretim amaçlı planladığınız Unreal Engine 5.7 sürümüne geçiş yaptığınızda, Arduino kodlarına hiç dokunmadan, sadece UE5 Blueprint'leri veya C++ Sockets üzerinden "A" ve "D" harflerini Serial Port'a yollayarak aynı robotları kontrol etmeye devam edebilirsiniz.
✨ Sürü (Swarm) Mantığı: ESP-NOW'ın asıl gücü birebir haberleşmede değil, "Çoklu Haberleşmede" yatar. 1 adet Master (Kumanda) ESP32'den çıkan sinyali aynı anda 20 farklı Alıcı ESP32'ye senkronize şekilde yollayabilirsiniz. Bu yapı ile gösteri amaçlı senkronize hareket eden robot sürüleri (Swarm Robotics) tasarlamak mümkündür.
