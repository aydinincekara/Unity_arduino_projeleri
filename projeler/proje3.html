Proje 3: Sanal Çarpışma, Gerçek Tepki

Unity'den Arduino'ya Veri Gönderme (Çift Yönlü İletişime Giriş)

🎯 Proje Hakkında

Bu projede veri akış yönünü tersine çeviriyoruz! Unity oyun motorunda oluşturduğumuz bir karakter, sahnedeki bir engele veya hedefe çarptığında (Collision/Trigger), Unity'den Arduino'ya bir sinyal göndereceğiz. Arduino bu sinyali aldığında fiziksel dünyadaki bir LED'i yakacak.

💡 Hap Bilgi: Unity'den Arduino'ya komut gönderirken karmaşık metinler yerine genellikle tek bir karakter (Örn: '1' veya '0') gönderilir. Bu, Arduino'nun gelen veriyi çok daha hızlı ve hatasız işlemesini sağlar.

🛠️ Gerekli Donanımlar ve Yazılımlar

Donanım Listesi

  • 1x Arduino UNO veya Nano
  • 1x Kırmızı LED
  • 1x 220 Ohm veya 330 Ohm Direnç
  • 1x Breadboard
  • Jumper Kablolar

Yazılım ve Kütüphaneler

  • IDE: Arduino IDE & Unity 3D
  • Unity Kütüphanesi: System.IO.Ports
  • Unity Kavramı: OnTriggerEnter / OnTriggerExit

🔌 Bağlantı Şeması

Fiziksel çıktımızı alacağımız uyarı LED'inin bağlantılarını aşağıdaki gibi yapınız:

Bileşen Pini Arduino Pini Açıklama
LED Uzun Bacak (Anot +) D8 (Digital 8) Sinyal ve 5V gücünü alacak bacak
LED Kısa Bacak (Katot -) GND (Direnç Üzerinden) Kısa bacağı 220 Ohm direnç ile GND'ye bağlayın (LED'in yanmaması için)
⚠️ Dikkat Edilmesi Gerekenler: Unity "Play" modundayken, oyun arka planda Arduino'nun seri portunu işgal eder (kilitler). Bu yüzden Unity çalışırken Arduino IDE üzerinden "Serial Monitor" ekranını açmaya çalışırsanız "Port Meşgul / Port Access Denied" hatası alırsınız.

⚙️ Nasıl Yapılır? (Adım Adım)

Adım 1: Arduino Kodunu Yükleme

Arduino'nun gelen veriyi "dinlemesi" için aşağıdaki kodu karta yükleyin.

const int ledPin = 8; // LED'i bağladığımız pin

void setup() {
  Serial.begin(9600); // Unity ile aynı hızda başlat
  pinMode(ledPin, OUTPUT); // LED pinini çıkış olarak ayarla
}

void loop() {
  // Eğer seri porttan gelen bir veri varsa:
  if (Serial.available() > 0) {
    char gelenVeri = Serial.read(); // Gelen karakteri oku
    
    // Eğer gelen karakter '1' ise LED'i yak
    if (gelenVeri == '1') {
      digitalWrite(ledPin, HIGH);
    } 
    // Eğer gelen karakter '0' ise LED'i söndür
    else if (gelenVeri == '0') {
      digitalWrite(ledPin, LOW);
    }
  }
}

Adım 2: Unity Tarafında Veriyi Gönderme

Unity sahnenizde bir oyuncu nesnesi ve bir hedef duvar oluşturun. Hedef duvarın Box Collider bileşeninde "Is Trigger" kutucuğunu işaretleyin. Ardından aşağıdaki C# betiğini oyuncu nesnesine (veya duvara) atayın.

using UnityEngine;
using System.IO.Ports;

public class CarpismaSinyali : MonoBehaviour {
    SerialPort port = new SerialPort("COM3", 9600); 

    void Start() {
        // Port kapalıysa aç
        if(!port.IsOpen) {
            port.Open();
        }
    }

    // Başka bir objenin Trigger alanına GİRDİĞİNDE çalışır
    void OnTriggerEnter(Collider other) {
        if(other.gameObject.tag == "HedefDuvar") {
            // Arduino'ya '1' karakterini gönder (LED'i yak)
            if(port.IsOpen) {
                port.Write("1");
                Debug.Log("Çarpışma gerçekleşti, LED yakılıyor!");
            }
        }
    }

    // Trigger alanından ÇIKTIĞINDA çalışır
    void OnTriggerExit(Collider other) {
        if(other.gameObject.tag == "HedefDuvar") {
            // Arduino'ya '0' karakterini gönder (LED'i söndür)
            if(port.IsOpen) {
                port.Write("0");
                Debug.Log("Alandan çıkıldı, LED söndürülüyor.");
            }
        }
    }

    void OnApplicationQuit() {
        if(port.IsOpen) {
            // Oyun kapanırken LED'in açık kalmaması için sıfırla
            port.Write("0");
            port.Close();
        }
    }
}

🚀 Tavsiyeler ve Basit Öneriler

✨ Geliştirme Önerisi 1: Bu devreyi sadece bir LED ile sınırlı bırakmayın. 8 numaralı pine bir LED yerine Transistör devresi üzerinden bir titreşim motoru (Haptic Motor) bağlarsanız, karakter duvara çarptığında tıpkı PlayStation oyun kollarında (gamepad) olduğu gibi fiziksel bir geri bildirim (titreşim) elde edersiniz.
✨ Geliştirme Önerisi 2: Projeyi daha büyük çaplı bir hale getirmek isterseniz, D8 pinine bir Röle (Relay) Modülü bağlayıp masa lambanızı kontrol edebilirsiniz. Unity'de gece olduğunda gerçek masanızın lambası otomatik yanabilir!
