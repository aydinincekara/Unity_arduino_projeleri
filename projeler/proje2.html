Proje 2: Akıllı Kontrolcü

Potansiyometre ve Buton ile 3D Nesne Manipülasyonu

🎯 Proje Hakkında

Bu projede, fiziksel dünyadaki çevresel bir hareketin (potansiyometreyi çevirme) dijital dünyadaki bir 3D nesnenin rotasyonunu (dönüşünü) nasıl etkilediğini göreceğiz. Ayrıca bir fiziksel butona basıldığında Unity içerisindeki nesnenin rengini veya boyutunu değiştirmeyi öğreneceğiz.

💡 Hap Bilgi: Analog sensörler (potansiyometre gibi) Arduino'da 0 ile 1023 arasında değer üretir. Bu değeri Unity'de 0 ile 360 derece arasına dönüştürerek (Mapping) tam bir tur dönüş sağlayacağız.

🛠️ Gerekli Donanımlar ve Yazılımlar

Donanım Listesi

  • 1x Arduino UNO veya Nano
  • 1x 10K Ohm Potansiyometre
  • 1x Push Buton (Yaylı Buton)
  • 1x Breadboard
  • Jumper Kablolar (Erkek-Erkek)

Yazılım ve Kütüphaneler

  • IDE: Arduino IDE & Unity 3D
  • Unity Kütüphanesi: System.IO.Ports
  • Arduino Kütüphanesi: Ekstra kütüphane gerekmez. Standart Serial kütüphanesi kullanılacaktır.

🔌 Bağlantı Şeması

Devrenizi kurarken aşağıdaki pin haritalandırmasını birebir uyguladığınızdan emin olun:

Bileşen Pini Arduino Pini Açıklama
Potansiyometre Sol Bacak 5V Güç girişi
Potansiyometre Orta Bacak A0 (Analog 0) Veri okuma (Sinyal)
Potansiyometre Sağ Bacak GND Topraklama
Buton 1. Bacak D2 (Digital 2) Kod içinde INPUT_PULLUP ile kullanılacak
Buton 2. Bacak GND Topraklama
⚠️ Dikkat Edilmesi Gerekenler: Buton bağlantısında harici bir direnç kullanmıyoruz. Bunun yerine Arduino'nun içindeki dahili direnci aktif etmek için kodda pinMode(2, INPUT_PULLUP); komutunu kullanmalıyız. Bu durumda buton boşta iken "1", basıldığında "0" değeri verir.

⚙️ Nasıl Yapılır? (Adım Adım)

Adım 1: Arduino Kodunu Yükleme

Devre bağlantılarını yaptıktan sonra aşağıdaki kodu Arduino'ya yükleyin. Bu kod, verileri X,Y formatında seri porta yazdırır.

void setup() {
  Serial.begin(9600);
  pinMode(2, INPUT_PULLUP); // Dahili direnci aktif et
}

void loop() {
  int potDeger = analogRead(A0);
  int butonDurum = digitalRead(2);

  // Verileri virgülle birleştirip gönderiyoruz
  Serial.print(potDeger);
  Serial.print(",");
  Serial.println(butonDurum);

  delay(20); // Dalgalanmayı önlemek için kısa bekleme
}

Adım 2: Unity Tarafında Veriyi İşleme

Unity'de sahnede bir 3D Küp oluşturun ve aşağıdaki C# betiğini bu küpe atayın.

using UnityEngine;
using System.IO.Ports;

public class ObjeKontrol : MonoBehaviour {
    SerialPort port = new SerialPort("COM3", 9600); // Kendi portunuzu yazın

    void Start() {
        port.Open();
        port.ReadTimeout = 50;
    }

    void Update() {
        if (port.IsOpen) {
            try {
                string veri = port.ReadLine();
                string[] parcalar = veri.Split(',');
                
                if(parcalar.Length == 2) {
                    float potVeri = float.Parse(parcalar[0]);
                    int butonVeri = int.Parse(parcalar[1]);

                    // Potansiyometre 0-1023 arası değer verir. Bunu 0-360 dereceye çeviriyoruz.
                    float aci = (potVeri / 1023f) * 360f;
                    transform.rotation = Quaternion.Euler(0, aci, 0);

                    // Butona basıldığında (0 olduğunda) objeyi büyüt
                    if(butonVeri == 0) {
                        transform.localScale = new Vector3(2, 2, 2);
                    } else {
                        transform.localScale = new Vector3(1, 1, 1);
                    }
                }
            } catch (System.Exception) { }
        }
    }

    void OnApplicationQuit() {
        if(port.IsOpen) port.Close();
    }
}

🚀 Tavsiyeler ve Basit Öneriler

✨ Geliştirme Önerisi 1: Unity sahnesinde kameranın arkasına güzel bir arka plan ekleyip, döndürdüğünüz 3D modeli kendi tasarladığınız bir uzay gemisine dönüştürebilirsiniz.
✨ Geliştirme Önerisi 2: Potansiyometreden gelen değerler bazen fiziksel titremelerden dolayı Unity ekranında pırpır (titreme) yapabilir. C# içerisinde Mathf.Lerp() fonksiyonu kullanarak bu dönüşleri yağ gibi pürüzsüz hale getirebilirsiniz.
