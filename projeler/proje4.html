Proje 4: Sanal Arayüz, Fiziksel Hareket

Unity UI (Arayüz) Elementleri ile Servo Motor Kontrolü

🎯 Proje Hakkında

Bir önceki projede Arduino'ya sadece "1" veya "0" göndererek bir ışığı açıp kapatmıştık. Şimdi ise çok daha dinamik bir veri ileteceğiz. Unity içerisinde tasarlayacağımız bir Kaydırma Çubuğunun (Slider) anlık değerini alıp, bu değeri fiziksel bir Servo motorun dönüş açısı (0-180 derece) olarak Arduino'ya komutlayacağız.

💡 Hap Bilgi: Servo motorlar PWM (Darbe Genişlik Modülasyonu) sinyalleri ile çalışır. Bu nedenle motorun sinyal kablosunu Arduino üzerinde yanında dalga işareti (~) bulunan özel pinlerden birine bağlamamız şarttır.

🛠️ Gerekli Donanımlar ve Yazılımlar

Donanım Listesi

  • 1x Arduino UNO veya Nano
  • 1x SG90 Mikro Servo Motor
  • Jumper Kablolar (Erkek-Erkek)

Yazılım ve Kütüphaneler

  • IDE: Arduino IDE & Unity 3D
  • Arduino Kütüphanesi: <Servo.h> (Arduino IDE ile gömülü gelir)
  • Unity Kütüphaneleri: UnityEngine.UI, System.IO.Ports

🔌 Bağlantı Şeması

Servo motorların genelde 3 kablosu bulunur (Kahverengi, Kırmızı ve Turuncu). Bağlantıları şu şekilde yapın:

Bileşen Pini Arduino Pini Açıklama
Kahverengi (veya Siyah) Kablo GND Topraklama / Eksi kutup
Kırmızı Kablo 5V Güç girişi / Artı kutup
Turuncu (veya Sarı) Kablo D9 (Digital 9) Veri (Sinyal) aktarımı. PWM pini olması önemlidir.
⚠️ Dikkat Edilmesi Gerekenler: Eğer projede birden fazla veya daha güçlü (Örn: MG996R) bir servo motor kullanırsanız, Arduino'nun 5V çıkışı yetersiz kalabilir ve kart kendini yeniden başlatabilir. Bu durumda motorlar için harici bir güç kaynağı (Örn: 4 adet AA Pil) kullanmalısınız.

⚙️ Nasıl Yapılır? (Adım Adım)

Adım 1: Arduino Kodunu Yükleme

Aşağıdaki kod, Unity'den gelen metin (String) formatındaki veri paketini okur, tam sayıya (Integer) çevirir ve bunu motor açısı olarak uygular.

#include <Servo.h>

Servo benimServom; // Servo nesnemizi oluşturuyoruz

void setup() {
  Serial.begin(9600); 
  benimServom.attach(9); // Servo motorun D9 pinine bağlı olduğunu belirtiyoruz
  benimServom.write(90);  // Başlangıçta tam ortaya (90 derece) hizala
}

void loop() {
  // Eğer seri porttan veri geliyorsa...
  if (Serial.available() > 0) {
    // Gelen veriyi '\n' (alt satır) karakterini görene kadar oku
    String gelenMetin = Serial.readStringUntil('\n'); 
    
    // Gelen metni tam sayıya (int) dönüştür
    int aciDegeri = gelenMetin.toInt();
    
    // Güvenlik: Değerin 0 ile 180 arasında olduğundan emin ol
    if(aciDegeri >= 0 && aciDegeri <= 180) {
      benimServom.write(aciDegeri);
    }
  }
}

Adım 2: Unity Tarafında Arayüzü Hazırlama

Unity'de üst menüden GameObject -> UI -> Slider yolunu izleyerek ekrana bir kaydırma çubuğu ekleyin. Slider'ın ayarlarından (Inspector paneli) Min Value: 0 ve Max Value: 180 olarak ve Whole Numbers: İşaretli olarak ayarlayın. Ardından aşağıdaki betiği sahnede boş bir GameObject'e atayın ve Slider'ı scriptin içine sürükleyin.

using UnityEngine;
using UnityEngine.UI;
using System.IO.Ports;

public class ServoKontrol : MonoBehaviour {
    SerialPort port = new SerialPort("COM3", 9600); // Port numaranızı ayarlayın
    public Slider aciSlider; // Unity arayüzündeki kaydırma çubuğu

    void Start() {
        if(!port.IsOpen) {
            port.Open();
        }
        
        // Slider değeri her değiştiğinde AciyiGonder fonksiyonunu otomatik çalıştır
        aciSlider.onValueChanged.AddListener(delegate { AciyiGonder(); });
    }

    void AciyiGonder() {
        if(port.IsOpen) {
            // Slider değerini tam sayıya çevir
            int aci = (int)aciSlider.value; 
            
            // Veriyi Arduino'ya gönder ve sonuna gizli '\n' (alt satır) karakteri ekle
            port.WriteLine(aci.ToString());
        }
    }

    void OnApplicationQuit() {
        if(port.IsOpen) port.Close();
    }
}

🚀 Tavsiyeler ve Basit Öneriler

✨ Geliştirme Önerisi 1: Bu projeyi bir araba yarışı simülasyonuna entegre edebilirsiniz! Unity'deki aracınızın o anki hız değişkenini alıp (örn: 0-180 km/h arası) bunu doğrudan servoya yollarsanız, masanızda çalışan fiziksel bir hız göstergesi (kadran) yapmış olursunuz.
✨ Geliştirme Önerisi 2: Servo milinin ucuna kartondan bir ok işareti yapıştırarak hareketin çok daha net ve eğlenceli bir şekilde gözlemlenmesini sağlayabilirsiniz.
