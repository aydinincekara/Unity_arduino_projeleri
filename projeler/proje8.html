Proje 8: Fiziksel Anahtar, Dijital Kilit

RC522 RFID Modülü ile Unity Kimlik Doğrulama ve Güvenlik Sistemi

🎯 Proje Hakkında

Sanal dünyada yer alan kilitli bir kapıyı, bir sunucu terminalini veya oyun içi bir sandığı açmak için klavye şifresi girmek yerine cebinizdeki gerçek bir RFID kartı kullanmaya ne dersiniz? Bu projede RC522 RFID okuyucu üzerinden kartın UID (Eşsiz Kimlik) kodunu okuyup Unity'ye aktaracağız. Unity bu kodu kendi veri tabanındaki (kod içindeki) yetkili kodlarla karşılaştırıp sistem erişimi verecek.

💡 Hap Bilgi: RC522 modülü, Arduino ile bir önceki projede kullandığımız I2C yerine SPI (Serial Peripheral Interface) adı verilen farklı ve çok daha hızlı bir protokol üzerinden haberleşir. Bu yüzden bağlantı pinleri özeldir (MISO, MOSI, SCK).

🛠️ Gerekli Donanımlar ve Yazılımlar

Donanım Listesi

  • 1x Arduino UNO veya Nano
  • 1x RC522 RFID Okuyucu Modülü
  • 1x 13.56MHz RFID Kart veya Anahtarlık (Tag)
  • Jumper Kablolar (Dişi-Erkek)

Yazılım ve Kütüphaneler

  • Arduino Kütüphanesi: MFRC522.h (Kütüphane yöneticisinden indirilmelidir) ve SPI.h
  • Unity Kütüphanesi: System.IO.Ports

🔌 Bağlantı Şeması

SPI haberleşmesi için Arduino UNO ve Nano'da pinler sabittir. Lütfen aşağıdaki şemaya tam uyum sağlayın:

RC522 Pini Arduino Pini Açıklama
3.3V 3.3V ÇOK ÖNEMLİ! Modül 5V ile bozulur, kesinlikle 3.3V pinine bağlanmalıdır.
RST (Reset) D9 Yeniden başlatma pini
GND GND Topraklama
MISO D12 Master In Slave Out (Gelen Veri Hattı)
MOSI D11 Master Out Slave In (Giden Veri Hattı)
SCK D13 Serial Clock (Saat Darbesi Hattı)
SDA (SS) D10 Slave Select (Çip Seçim Pini)
⚠️ Dikkat Edilmesi Gerekenler: Modül 3.3V ile çalışır. VCC veya 5V pinine takarsanız RC522 modülü kalıcı olarak yanabilir. Veri pinleri 5V toleranslıdır (direkt D10-D13'e takılabilir), ancak güç bağlantısı kesinlikle 3.3V olmalıdır.

⚙️ Nasıl Yapılır? (Adım Adım)

Adım 1: Arduino Kodunu Yükleme

Kütüphane Yöneticisinden "MFRC522 by GithubCommunity" kütüphanesini kurun ve aşağıdaki kodu yükleyin. Bu kod, okunan kartın kimliğini (Örn: A1 B2 C3 D4) aralarında boşluk bırakmadan (Örn: A1B2C3D4) Unity'ye gönderir.

#include <SPI.h>
#include <MFRC522.h>

#define SS_PIN 10
#define RST_PIN 9

MFRC522 rfid(SS_PIN, RST_PIN);

void setup() {
  Serial.begin(9600);
  SPI.begin(); // SPI veriyolunu başlat
  rfid.PCD_Init(); // RFID okuyucuyu başlat
}

void loop() {
  // Yeni kart okutulmadıysa başa dön
  if (!rfid.PICC_IsNewCardPresent() || !rfid.PICC_ReadCardSerial()) {
    return;
  }

  String kartUID = "";
  // 4 byte'lık UID kodunu oku ve Hexadecimal (16'lık) sisteme çevir
  for (byte i = 0; i < rfid.uid.size; i++) {
    kartUID += String(rfid.uid.uidByte[i] < 0x10 ? "0" : "");
    kartUID += String(rfid.uid.uidByte[i], HEX);
  }

  kartUID.toUpperCase(); // Tüm harfleri büyüt
  
  // Sadece UID kodunu seri porta yazdır
  Serial.println(kartUID);

  // Aynı kartın defalarca okunmasını engellemek için kartı uykuya al
  rfid.PICC_HaltA();
}

Adım 2: Unity Tarafında Veriyi İşleme

Unity'de sisteme giriş yapılmasını temsil eden bir "Yetkili Girişi" arayüzü veya sanal bir kapı (Door) oluşturun ve bu kodu atayın. yetkiliKartUID kısmına, kendi kartınızın UID kodunu yazmayı unutmayın (Bunu Arduino Serial Monitor'den öğrenebilirsiniz).

using UnityEngine;
using System.IO.Ports;

public class RFIDGuvenlik : MonoBehaviour {
    SerialPort port = new SerialPort("COM3", 9600); 
    
    // Geçerli kartın kodunu buraya yazın (Örn: 23A5B802)
    public string yetkiliKartUID = "KENDI_KART_KODUNUZU_YAZIN";

    void Start() {
        if(!port.IsOpen) port.Open();
        port.ReadTimeout = 50;
    }

    void Update() {
        if (port.IsOpen) {
            try {
                // Arduino'dan gelen kart okuma verisini yakala
                string okunanKart = port.ReadLine().Trim(); 
                
                if (!string.IsNullOrEmpty(okunanKart)) {
                    Debug.Log("Okunan Kart UID: " + okunanKart);
                    
                    if (okunanKart == yetkiliKartUID) {
                        ErisimOnaylandi();
                    } else {
                        ErisimReddedildi();
                    }
                }
            } 
            catch (System.TimeoutException) { /* Gecikmeleri yoksay */ }
        }
    }

    void ErisimOnaylandi() {
        Debug.Log("🟢 ERIŞIM ONAYLANDI! Kilit Açılıyor...");
        // Burada kapı animasyonunu çalıştırabilir veya sahne geçişi yapabilirsiniz.
        // GetComponent<Animator>().SetTrigger("KapiAc");
    }

    void ErisimReddedildi() {
        Debug.Log("🔴 YETKISIZ KART! Giriş Reddedildi.");
        // Burada bir hata sesi (AudioSource.Play) çalabilirsiniz.
    }

    void OnApplicationQuit() {
        if(port.IsOpen) port.Close();
    }
}

🚀 Tavsiyeler ve Basit Öneriler

✨ Geliştirme Önerisi 1: Siber Güvenlik Masası Uygulaması: Fiziksel bir sunucu masası (Network Topology Desk) maketi yapıp üzerine bu RFID okuyucuları yerleştirebilirsiniz. Öğrenciler "Yetkilendirme Kartı"nı (RFID) doğru istasyona okuttuklarında Unity ekranındaki ağ haritasında ilgili sanal sunucunun (Node) kilidi açılabilir veya bir siber saldırı engellenebilir.
✨ Geliştirme Önerisi 2: Projeyi çift yönlü hale getirmek için 3. projedeki LED mantığını ekleyebilirsiniz. Doğru kart okutulduğunda Unity Arduino'ya '1' göndersin ve masadaki "Yeşil LED" yansın, yanlış kartta '0' göndersin ve "Kırmızı LED" yansın.
