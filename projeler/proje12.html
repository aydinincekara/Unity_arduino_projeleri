Proje 12: Gizli Şifreler (Şifre Paneli)

Membran Keypad ile Unity'de Kilit ve Durum Makinesi Kontrolü

🎯 Proje Hakkında

Kaçış odalarında (Escape Room) veya macera oyunlarında gizli kasaları açmak harika bir histir! Bu projede 4x3 boyutunda bir membran tuş takımı kullanacağız. Oyuncu tuşlara bastıkça Unity tarafında bir "Şifre Dizisi (String)" oluşacak. Oyuncu belirlediğimiz 4 haneli şifreyi doğru girdiğinde sanal kilit açılacak, yanlış girdiğinde ise kırmızı bir uyarı vererek sistem sıfırlanacak.

💡 Hap Bilgi (Matris Tarama): Keypad'in içinde her tuş için ayrı bir kablo yoktur. 12 tuş için 7 kablo (4 Satır, 3 Sütun) bulunur. Arduino, satırlara sırayla elektrik verip sütunlardan okuma yaparak (Matris Tarama Yöntemi) hangi tuşa basıldığını koordinat düzlemi gibi tespit eder.

🛠️ Gerekli Donanımlar ve Yazılımlar

Donanım Listesi

  • 1x Arduino UNO veya Nano
  • 1x 4x3 Membran Keypad (Tuş Takımı)
  • Jumper Kablolar (Erkek-Erkek)

Yazılım ve Kütüphaneler

  • Arduino IDE: Keypad.h kütüphanesi (Kütüphane yöneticisinden indirilmelidir).
  • Unity: System.IO.Ports ve UI metin güncellemeleri için UnityEngine.UI (veya TextMeshPro).

🔌 Bağlantı Şeması

Membran keypad'in altından çıkan 7 adet pin (veya 4x4 ise 8 pin) bulunur. Soldan sağa doğru ilk 4 pin Satırları (Rows), son 3 pin ise Sütunları (Cols) temsil eder.

Keypad Pini (Soldan Sağa) Görevi Arduino Pini
Pin 1 (En Sol)Satır 1 (R1)D9
Pin 2Satır 2 (R2)D8
Pin 3Satır 3 (R3)D7
Pin 4Satır 4 (R4)D6
Pin 5Sütun 1 (C1)D5
Pin 6Sütun 2 (C2)D4
Pin 7 (En Sağ)Sütun 3 (C3)D3
⚠️ Dikkat Edilmesi Gerekenler: Güç (5V) veya GND bağlantısı YAPMIYORUZ! Keypad sadece dijital pinlere bağlanır, elektriği Arduino kütüphanesi otomatik yönetir.

⚙️ Nasıl Yapılır? (Adım Adım)

Adım 1: Arduino Kodunu Yükleme

Arduino Kütüphane Yöneticisinden Keypad by Mark Stanley kütüphanesini kurun. Aşağıdaki kod, basılan her tuşu tek bir karakter olarak (Örn: '5') Unity'ye gönderir.

#include <Keypad.h>

const byte ROWS = 4; // 4 Satır
const byte COLS = 3; // 3 Sütun

// Tuş takımının haritasını çıkarıyoruz
char keys[ROWS][COLS] = {
  {'1','2','3'},
  {'4','5','6'},
  {'7','8','9'},
  {'*','0','#'}
};

// Keypad pinlerini Arduino pinleriyle eşleştiriyoruz (Tablodaki sırayla)
byte rowPins[ROWS] = {9, 8, 7, 6}; 
byte colPins[COLS] = {5, 4, 3}; 

// Keypad nesnesini oluştur
Keypad keypad = Keypad(makeKeymap(keys), rowPins, colPins, ROWS, COLS);

void setup() {
  Serial.begin(9600);
}

void loop() {
  // Basılan tuşu oku
  char key = keypad.getKey();

  // Eğer bir tuşa basıldıysa...
  if (key) {
    Serial.println(key); // Tuşu Unity'ye gönder
  }
}

Adım 2: Unity Tarafında Veriyi İşleme

Unity'de sahnede bir Kapı objesi ve ekranda (Canvas) şifreyi gösterecek bir Text (UI) oluşturun. Aşağıdaki betiği boş bir objeye atayın.

using UnityEngine;
using UnityEngine.UI;
using System.IO.Ports;

public class SifreliKapi : MonoBehaviour {
    SerialPort port = new SerialPort("COM3", 9600);
    
    [Header("Şifre Ayarları")]
    public string dogruSifre = "1453"; // Hedef şifremiz
    private string girilenSifre = "";  // Oyuncunun o an girdiği şifre hafızası
    
    [Header("Arayüz (UI)")]
    public Text ekranMetni; // Unity'deki UI Text objesi

    void Start() {
        if(!port.IsOpen) port.Open();
        port.ReadTimeout = 50;
        EkranGuncelle("Sifre Girin");
    }

    void Update() {
        if (port.IsOpen) {
            try {
                // Arduino'dan gelen TEK KARAKTERLİ stringi oku
                string gelenTus = port.ReadLine().Trim();

                if (!string.IsNullOrEmpty(gelenTus)) {
                    TusaBasildi(gelenTus);
                }
            } 
            catch (System.TimeoutException) { /* Gecikmeleri yoksay */ }
        }
    }

    void TusaBasildi(string tus) {
        // Eğer '*' tuşuna basılırsa şifreyi sıfırla (Silme Tuşu)
        if (tus == "*") {
            girilenSifre = "";
            EkranGuncelle("Sifirlandi");
            return;
        }

        // Eğer '#' tuşuna basılırsa işlemi onayla (Enter Tuşu)
        if (tus == "#") {
            SifreyiKontrolEt();
            return;
        }

        // 4 haneden fazla girilmesini engelle
        if (girilenSifre.Length < 4) {
            girilenSifre += tus; // Yeni tuşu mevcut şifrenin sonuna ekle
            EkranGuncelle(girilenSifre);
            
            // 4 hane dolduğu an otomatik kontrol et
            if(girilenSifre.Length == 4) {
                SifreyiKontrolEt();
            }
        }
    }

    void SifreyiKontrolEt() {
        if (girilenSifre == dogruSifre) {
            EkranGuncelle("KILIT ACILDI!");
            Debug.Log("Başarılı! Kapı Açılıyor...");
            // GetComponent<Animator>().SetTrigger("KapiAc");
        } else {
            EkranGuncelle("HATALI SIFRE!");
            Debug.Log("Hata! Yanlış Şifre.");
            girilenSifre = ""; // Şifreyi hatalı girdikten sonra hafızayı temizle
        }
    }

    void EkranGuncelle(string mesaj) {
        if(ekranMetni != null) {
            ekranMetni.text = mesaj;
        }
    }

    void OnApplicationQuit() {
        if(port.IsOpen) port.Close();
    }
}

🚀 Tavsiyeler ve Basit Öneriler

✨ Geliştirme Önerisi 1: Görsel ve İşitsel Geri Bildirim: Oyuncu tuşlara her bastığında Unity içerisinden kısa bir 'Bip' sesi (AudioSource.PlayOneShot) çalın. Şifre doğru girildiğinde ekran metni (UI Text) Yeşil renge, yanlış girildiğinde ise Kırmızı renge dönecek şekilde kodunuzu modifiye ederseniz proje çok daha profesyonel hissettirir.
✨ Geliştirme Önerisi 2: Durum Makinesi (State Machine): Şifre sistemini bir "Envanter / Silah Seçim" menüsüne dönüştürebilirsiniz. Örneğin Unity'de '1' tuşuna basılınca karakteriniz kılıcını çeksin, '2' tuşuna basıldığında ok ve yayı çıkarsın. Keypad'ler oyunlarda muazzam bir "Kısayol Paneli (Macro Pad)" olarak kullanılabilir!
