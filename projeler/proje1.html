Eğitim Rehberi: Unity ve Arduino Entegrasyonu

3D Baskı Destekli Simülatör Kontrolcüsü Geliştirme

📌 Proje Hakkında

Bu rehber, fiziksel donanım ile dijital ortam (Unity) arasındaki seri haberleşme mantığını kavramak amacıyla hazırlanmıştır. Genç yetişkinler ve lise kademesi için yapılandırmacı (constructivist) öğrenme yaklaşımıyla ve sarmal müfredat mantığına uygun olarak tasarlanmıştır. Öğrenciler soyut yazılım kavramlarını, kendi ürettikleri somut bir donanım üzerinden deneyimleyerek öğreneceklerdir.

1 İhtiyaç Listesi ve 3D Baskı Hazırlığı

Simülatör kasasını oluştururken STL veya 3MF formatındaki dosyalarınızı dilimleyicide hazırlayıp 3D yazıcılarınızda üretebilirsiniz. Kasaya yerleştirilecek bileşenler şunlardır:

  • Mikrodenetleyici: Arduino Nano (Kompakt yapısı nedeniyle kasaya kolay gizlenir).
  • Eksen Kontrolü: 1x İki Eksenli Joystick Modülü.
  • Gaz/Hız Kontrolü: 1x 10K Ohm Potansiyometre.
  • Aksiyon: 2x Yaylı Buton (Push Button) veya Havacılık Tipi Switch.
  • Bağlantı ve Montaj: Breadboard (prototipleme için), daha sonra kalıcı sistem için JST kablolar ve kurşunsuz lehim teli kullanımı önerilir.

2 Arduino (C++) Kodlaması ve Veri Gönderimi

Aşağıdaki kod, sensörlerden okunan değerleri tek bir metin satırı halinde aralarına virgül koyarak bilgisayara gönderir.

// Sensör pin tanımlamaları
const int joyXPin = A0;       // Joystick X Ekseni
const int joyYPin = A1;       // Joystick Y Ekseni
const int potPin = A2;        // Gaz (Potansiyometre) Pini
const int buttonPin = 2;      // Fonksiyon Butonu Pini

void setup() {
  // Seri haberleşmeyi başlat (Unity ile aynı hızda olmalı)
  Serial.begin(9600);
  
  // Dahili pull-up direncini aktif ediyoruz (Boşta 1, basılınca 0 döner)
  pinMode(buttonPin, INPUT_PULLUP);
}

void loop() {
  // Değerleri oku (Analog 0-1023 arası)
  int joyX = analogRead(joyXPin);
  int joyY = analogRead(joyYPin);
  int pot = analogRead(potPin);
  int buttonState = digitalRead(buttonPin); 

  // Verileri paketle ve gönder
  Serial.print(joyX);
  Serial.print(",");
  Serial.print(joyY);
  Serial.print(",");
  Serial.print(pot);
  Serial.print(",");
  Serial.println(buttonState); // Satır sonu karakteri (NewLine) ile bitir

  delay(20); // Kararlı akış için ~50Hz yenileme hızı
}

3 Unity (C#) Kodlaması ve Veri Çözümleme (Parsing)

Unity'de boş bir GameObject oluşturun ve aşağıdaki C# betiğini ekleyin. Unity'de Edit -> Project Settings -> Player -> API Compatibility Level ayarının .NET Framework olduğundan emin olun (System.IO.Ports kullanımı için).

using System;
using System.IO.Ports;
using UnityEngine;

public class SimulatorController : MonoBehaviour
{
    private SerialPort _serialPort;
    
    [Header("Bağlantı Ayarları")]
    public string portName = "COM3"; // Aygıt yöneticisinden kontrol edin
    public int baudRate = 9600;

    [Header("Simülasyon Ayarları")]
    public Transform targetObject;   // Hareket ettirilecek 3D model
    public float rotationSpeed = 15f;

    void Start()
    {
        _serialPort = new SerialPort(portName, baudRate);
        _serialPort.ReadTimeout = 50; 

        try
        {
            _serialPort.Open();
            Debug.Log("Port açıldı: " + portName);
        }
        catch (Exception e)
        {
            Debug.LogError("Bağlantı Hatası: " + e.Message);
        }
    }

    void Update()
    {
        if (_serialPort != null && _serialPort.IsOpen)
        {
            try
            {
                string incomingData = _serialPort.ReadLine();
                if (!string.IsNullOrEmpty(incomingData))
                {
                    ParseData(incomingData);
                }
            }
            catch (TimeoutException) { /* Veri bekleniyor */ }
        }
    }

    void ParseData(string data)
    {
        string[] splitValues = data.Split(',');

        if (splitValues.Length >= 4)
        {
            try
            {
                int joyX = int.Parse(splitValues[0]);
                int joyY = int.Parse(splitValues[1]);
                int pot = int.Parse(splitValues[2]);
                int button = int.Parse(splitValues[3]);

                // 0-1023 verisini -1 ile +1 aralığına dönüştür
                float moveX = Mathf.InverseLerp(0, 1023, joyX) * 2 - 1;
                float moveY = Mathf.InverseLerp(0, 1023, joyY) * 2 - 1;
                float throttle = Mathf.InverseLerp(0, 1023, pot);

                // 3D Modele Uygula
                if (targetObject != null)
                {
                    Vector3 rot = new Vector3(moveY * rotationSpeed, 0, -moveX * rotationSpeed);
                    targetObject.Rotate(rot * Time.deltaTime);
                }

                if (button == 0)
                {
                    Debug.Log("Aksiyon Butonuna Basıldı! Gaz: " + throttle);
                }
            }
            catch (FormatException) { /* Hatalı paketleri atla */ }
        }
    }

    void OnApplicationQuit()
    {
        // Unity kapandığında portu serbest bırak
        if (_serialPort != null && _serialPort.IsOpen)
        {
            _serialPort.Close();
        }
    }
}
