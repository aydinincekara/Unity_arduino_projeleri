Proje 1: İlk Adım

Fiziksel Buton ile Dijital Dünyaya Dokunmak (Karakter Zıplatma)

🎯 Proje Hakkında

Donanım ile yazılımı birleştirmenin en temel ve en keyifli yolu bir butona basmaktır! Bu ilk projemizde, masanızdaki fiziksel bir butona bastığınızda Unity sahnenizdeki bir karakterin (veya küpün) zıplamasını sağlayacağız. Klavyedeki "Boşluk" tuşunun görevini artık kendi kurduğumuz devre üstlenecek.

💡 Hap Bilgi: Dijital sensörler (Buton, Switch, Hareket Sensörü) sadece iki farklı değer üretir: 1 (HIGH / Açık) veya 0 (LOW / Kapalı).

🛠️ Gerekli Donanımlar ve Yazılımlar

Donanım Listesi

  • 1x Arduino UNO veya Nano
  • 1x Push Buton (Yaylı Buton)
  • 1x Breadboard
  • Jumper Kablolar

Yazılım ve Kütüphaneler

  • IDE: Arduino IDE & Unity 3D
  • Unity Kütüphanesi: System.IO.Ports
  • Unity Fizik Motoru: Rigidbody (Zıplama için gereklidir)

🔌 Bağlantı Şeması

Butonu bağlarken fazladan direnç kullanmamak adına Arduino'nun dahili direncini kullanacağız:

Bileşen Pini Arduino Pini Açıklama
Buton 1. Bacak D2 (Digital 2) Sinyali okuyacağımız pin.
Buton 2. Bacak GND (Toprak) Devreyi tamamlayan topraklama hattı.
⚠️ Dikkat Edilmesi Gerekenler: Unity'de zıplama kuvvetinin (AddForce) çalışabilmesi için, hareket ettireceğiniz objeye mutlaka Rigidbody bileşenini (Add Component -> Physics -> Rigidbody) eklemiş olmanız gerekmektedir.

⚙️ Nasıl Yapılır? (Adım Adım)

Adım 1: Arduino Kodunu Yükleme

Aşağıdaki kod, 2 numaralı pindeki butonun durumunu okur ve bunu seri port üzerinden bilgisayara iletir.

const int butonPin = 2; // Butonun bağlı olduğu pin

void setup() {
  Serial.begin(9600); // Unity ile aynı hız
  // Dahili direnci açarak kısa devre riskini sıfırlıyoruz:
  pinMode(butonPin, INPUT_PULLUP); 
}

void loop() {
  int butonDurumu = digitalRead(butonPin);
  
  // Sadece buton durumunu (0 veya 1) gönderip alt satıra geçiyoruz
  Serial.println(butonDurumu); 
  
  delay(10); // Arduino'yu yormamak için kısa bir bekleme
}

Adım 2: Unity Tarafında Veriyi İşleme

Unity sahnenizde bir zemin (Plane) ve üstüne bir Küp (Cube) oluşturun. Küpe Rigidbody ekleyin. Aşağıdaki C# betiğini oluşturup küpün içine sürükleyin.

using UnityEngine;
using System.IO.Ports;

public class ZiplaKontrol : MonoBehaviour {
    SerialPort port = new SerialPort("COM3", 9600); 
    public float ziplamaGucu = 300f; // Ne kadar yükseğe zıplayacağı
    private Rigidbody rb;
    private bool ziplayabilirMi = true; // Sürekli havaya uçmasını engellemek için

    void Start() {
        rb = GetComponent<Rigidbody>();
        
        if(!port.IsOpen) {
            port.Open();
            port.ReadTimeout = 50;
        }
    }

    void Update() {
        if (port.IsOpen) {
            try {
                // Arduino'dan gelen metni ("0" veya "1") oku
                string gelenVeri = port.ReadLine(); 
                
                // INPUT_PULLUP kullanıldığı için butona BASILDIĞINDA "0" gelir
                if (gelenVeri == "0" && ziplayabilirMi) {
                    Zipla();
                }
            } 
            catch (System.Exception) { /* Veri gecikmelerini yoksay */ }
        }
    }

    void Zipla() {
        // Objeye yukarı doğru bir fiziksel kuvvet uygula
        rb.AddForce(Vector3.up * ziplamaGucu);
        ziplayabilirMi = false; // Havadayken tekrar zıplamayı kitle
    }

    // Obje yere çarptığında tekrar zıplamaya izin ver
    void OnCollisionEnter(Collision temas) {
        if (temas.gameObject.name == "Plane" || temas.gameObject.name == "Zemin") {
            ziplayabilirMi = true; 
        }
    }

    void OnApplicationQuit() {
        if(port.IsOpen) port.Close();
    }
}

🚀 Tavsiyeler ve Basit Öneriler

✨ Geliştirme Önerisi 1: Sisteme birden fazla buton ekleyerek (Örneğin D2, D3, D4 pinlerine) karakterin sadece zıplamasını değil, aynı zamanda sağa ve sola gitmesini de sağlayabilirsiniz. Bunun için bir "Fiziksel Oyun Kumandası (Gamepad)" yapabilirsiniz!
✨ Geliştirme Önerisi 2: Kod içerisindeki ziplamaGucu = 300f; değerini Unity Inspector ekranından oyun oynanırken değiştirerek, karakterinizin ağırlığına en uygun zıplama hissiyatını bulabilirsiniz.
