Proje 9: Dijital İkiz (Digital Twin)

DHT11 Sensörü ile Unity'de Gerçek Zamanlı Hava Durumu Simülasyonu

🎯 Proje Hakkında

Gerçek dünyanın dijital bir kopyasını yaratmaya "Dijital İkiz" denir. Bu projede DHT11 Sıcaklık ve Nem Sensörünü kullanarak fiziksel ortamınızın hava verilerini Unity'ye aktaracağız. Bu verilerle Unity'nin Particle System (Parçacık Sistemi) motorunu kontrol edeceğiz. Odanızın nemi arttığında Unity'deki sanal ormanda yağmur yağmaya başlayacak!

💡 Hap Bilgi: DHT11 sensörü analog gibi görünse de aslında dijital bir sensördür. Tek bir kablo üzerinden hem sıcaklık hem de nem verisini özel bir bit dizilimiyle gönderir. Bu karmaşık veriyi çözümlemek için Arduino tarafında özel bir kütüphane (DHT.h) kullanmamız gerekir.

🛠️ Gerekli Donanımlar ve Yazılımlar

Donanım Listesi

  • 1x Arduino UNO veya Nano
  • 1x DHT11 Isı ve Nem Sensörü (Modül tipi tercih edilir)
  • 1x Breadboard
  • Jumper Kablolar

Yazılım ve Kütüphaneler

  • Arduino IDE: Adafruit DHT Sensor Library (Kütüphane yöneticisinden indirin)
  • Unity: System.IO.Ports ve yağmur efekti için ParticleSystem bileşeni.

🔌 Bağlantı Şeması

Eğer modül tipi bir DHT11 kullanıyorsanız (küçük bir PCB üzerine lehimli, 3 bacaklı), bağlantıları aşağıdaki gibi yapın:

DHT11 Modül Pini Arduino Pini Açıklama
+ veya VCC 5V Güç girişi
OUT veya DATA D7 (Digital 7) Veri aktarım pini
- veya GND GND Topraklama
⚠️ Dikkat Edilmesi Gerekenler: Modülsüz (çıplak, 4 bacaklı mavi kutu) bir DHT11 kullanıyorsanız, VCC ile DATA pini arasına mutlaka 10K ohm'luk bir Pull-Up direnci bağlamanız gerekir, aksi takdirde sensörden "Nan" (Not a Number) hatası alırsınız.

⚙️ Nasıl Yapılır? (Adım Adım)

Adım 1: Arduino Kodunu Yükleme

Adafruit'in "DHT sensor library" kütüphanesini kurduktan sonra bu kodu yükleyin. Kod sıcaklık ve nemi okuyup "Sıcaklık,Nem" formatında Unity'ye gönderecektir.

#include "DHT.h"

#define DHTPIN 7      // DHT11 Data pininin bağlı olduğu dijital pin
#define DHTTYPE DHT11 // Sensör tipini belirtiyoruz

DHT dht(DHTPIN, DHTTYPE);

void setup() {
  Serial.begin(9600);
  dht.begin(); // Sensörü başlat
}

void loop() {
  // DHT11 yavaş bir sensördür, okumalar arasında en az 2 saniye beklemeliyiz
  delay(2000);

  // Nemi (h) ve Sıcaklığı Celsius (t) cinsinden oku
  float h = dht.readHumidity();
  float t = dht.readTemperature();

  // Okuma başarısız olursa tekrar dene
  if (isnan(h) || isnan(t)) {
    Serial.println("Sensör okuma hatası!");
    return;
  }

  // Verileri virgülle paketle ve yolla (Örn: 24.50,60.00)
  Serial.print(t);
  Serial.print(",");
  Serial.println(h);
}

Adım 2: Unity Tarafında Veriyi İşleme

Unity sahnenizde GameObject -> Effects -> Particle System oluşturun (Bu sizin yağmur efektiniz olacak). Ardından boş bir GameObject'e aşağıdaki scripti atayıp, Particle System'i Inspector üzerinden scripte tanımlayın.

using UnityEngine;
using System.IO.Ports;

public class DijitalIkizHava : MonoBehaviour {
    SerialPort port = new SerialPort("COM3", 9600); 
    public ParticleSystem yagmurSistemi; // Inspector'dan sürükleyin
    public float yagmurBaslamaNemi = 65.0f; // Yağmurun başlayacağı nem yüzdesi

    void Start() {
        if(!port.IsOpen) port.Open();
        port.ReadTimeout = 50;

        // Oyun başlarken yağmuru durdur
        var emisyon = yagmurSistemi.emission;
        emisyon.enabled = false;
    }

    void Update() {
        if (port.IsOpen) {
            try {
                string gelenVeri = port.ReadLine();
                string[] sensorVerileri = gelenVeri.Split(',');

                if (sensorVerileri.Length == 2) {
                    float sicaklik = float.Parse(sensorVerileri[0], System.Globalization.CultureInfo.InvariantCulture);
                    float nem = float.Parse(sensorVerileri[1], System.Globalization.CultureInfo.InvariantCulture);

                    // Değerleri konsolda takip et
                    Debug.Log("Sıcaklık: " + sicaklik + "°C | Nem: %" + nem);

                    // Hava kontrol mekanizması
                    HavaDurumunuGuncelle(sicaklik, nem);
                }
            } 
            catch (System.Exception) { /* Gecikmeleri yoksay */ }
        }
    }

    void HavaDurumunuGuncelle(float anlikSicaklik, float anlikNem) {
        var emisyon = yagmurSistemi.emission;

        // Eğer fiziksel odadaki nem, belirlediğimiz sınırın üstündeyse
        if (anlikNem > yagmurBaslamaNemi) {
            emisyon.enabled = true; // Yağmuru başlat
        } else {
            emisyon.enabled = false; // Yağmuru durdur
        }
    }

    void OnApplicationQuit() {
        if(port.IsOpen) port.Close();
    }
}

🚀 Tavsiyeler ve Basit Öneriler

✨ Geliştirme Önerisi 1: Çift Yönlü İkiz: Dijital ikiz kavramını tamamlamak için geri bildirim ekleyin! Eğer Unity içindeki sıcaklık çok yükselirse (Örneğin yanardağ simülasyonu), Unity Arduino'ya sinyal göndersin ve masanızdaki küçük bir fan motoru (Röle üzerinden) çalışmaya başlasın!
✨ Geliştirme Önerisi 2: `System.Globalization.CultureInfo.InvariantCulture` kodu çok önemlidir. Arduino ondalık sayıları nokta (.) ile gönderirken, Türkçe Windows işletim sistemleri virgül (,) arar ve bu durum çöküşlere neden olabilir. Bu kod, formatlama hatalarını kalıcı olarak çözer.
